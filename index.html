Khmer Text Normalization Demo

To use the Khmer Text Normalization tool, simply enter Khmer text into the input field labeled "សូមសរសេរជាភាសាខ្មែរ".

Then, click the "Normalize" button to start the process. The cleaned and corrected text will appear in the output box, where you can easily copy or download it as a .txt file format.

If you'd like to normalize longer paragraphs, you can also upload a .txt file into "Load File" button for normalization.
